# 🌸 Iris Flower Prediction App

A Machine Learning web application that predicts the species of an Iris flower based on user input measurements.

The app is built using **Python**, **Scikit-learn**, and **Streamlit**, and deployed online so anyone can use it.

## 📌 Project Overview

The Iris Flower Prediction App allows users to input flower measurements and instantly predict the species of the flower.

The model predicts one of the following species:

* Setosa
* Versicolor
* Virginica

The application demonstrates how a machine learning model can be integrated with a web interface for real-time predictions.

---

## 🧠 Machine Learning Model

The model was trained using the famous Iris dataset.

Steps performed:

1. Data loading and preprocessing
2. Model training using Scikit-learn
3. Model serialization using Joblib
4. Deployment using Streamlit

---

## 📊 Input Features

The model uses four flower measurements:

| Feature      | Description              |
| ------------ | ------------------------ |
| Sepal Length | Length of the sepal (cm) |
| Sepal Width  | Width of the sepal (cm)  |
| Petal Length | Length of the petal (cm) |
| Petal Width  | Width of the petal (cm)  |

## ▶️ Run Locally

Clone the repository:

```
git clone https://github.com/yourusername/iris_app.git
```

Navigate to the project folder:

```
cd iris_app
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the Streamlit app:

```
streamlit run app.py
```
